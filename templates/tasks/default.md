## 验证要求

请根据项目情况选择最小有效验证方式。

优先使用项目已有命令，例如：

```bash
npm run build
npm test
npm run lint
```

如果没有现成命令，请选择合适的 smoke check，例如：

- 启动项目确认无报错。
- 执行脚本确认输出正确。
- 检查目标文件是否存在且内容正确。
- 手动检查关键页面或关键流程。

如果无法验证，请明确说明原因。
