# 路线图

Alice Coding 当前处于 `v0.1.x` 早期阶段，目标是先把本地闭环工作流跑稳，再逐步提升安装体验、跨平台能力和生态扩展。

## v0.1.x：本地闭环稳定

- 稳定 `vibe` 基础命令：初始化、导入文档、任务、运行、审查、学习、归档。
- 稳定四个 MCP server：skills、project-ops、verification、reference。
- 完善中文文档和模板。
- 补充 demo project 和 smoke test。
- 增强安装脚本的错误提示。
- 持续完善 skill 维护体验，包括 `vibe skill doctor/sync` 和 skill 优化任务模板。

## v0.2.x：安装和配置体验

- 提供更完整的一键配置向导。
- 自动生成 Codex / Claude Code MCP 配置片段。
- 支持检查 Node、Codex、Claude Code、MCP server 构建状态。
- 增加更多项目类型 recipe。

## v0.3.x：验证和参考能力增强

- 更智能的 verification selector。
- 支持更多语言和框架的验证矩阵。
- reference MCP 增加 license 提醒和风险标记。
- 支持更清晰的 run summary 和复盘报告。

## 长期方向

- 让每个项目拥有自己的本地 AI 操作层。
- 让 Agent 能更稳定地读需求、做计划、执行、验证和沉淀经验。
- 在保持本地优先和安全边界的前提下，支持更多 Agent、MCP 和 skills 生态。
